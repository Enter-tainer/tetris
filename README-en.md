# Tetris

[简体中文](README.md) | [English](README-en.md)

Modern tetris that runs on self-made RISC-V CPU and x86 machines.

- Super Rotation System
- bag 7 
- hold
- next preview
- T spin
- Versus mode！

## Build

```bash
CC=clang LD=lld cmake -B build
make
```
